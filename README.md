# PawClinic

A modern web application for managing a veterinary clinic. Built with the MERN stack (MongoDB, Express, React, Node) and styled with Tailwind CSS.

## Features
- Appointment booking and management.
- AI Assistant for quick queries.
- User authentication (Login/Signup).
- Responsive design.

## Tech Stack
- **Frontend:** React 19, Vite, Tailwind CSS 4.
- **Backend:** Node.js, Express, MongoDB (Mongoose).
- **AI:** Google Generative AI (Gemini).

## Setup Instructions

### Backend
1. Navigate to `backend` folder.
2. Run `npm install`.
3. Create a `.env` file based on `.env.example` (or use the one provided) and fill in your credentials.
4. Run `npm start` or `npm run dev`.

### Frontend
1. Navigate to `frontend` folder.
2. Run `npm install`.
3. Create a `.env` file and set `VITE_API_URL` to your backend URL.
4. Run `npm run dev` to start the development server.

## Deployment
- **Frontend:** Can be deployed on Vercel or Netlify.
- **Backend:** Can be deployed on Render or Railway.
